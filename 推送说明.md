# 推送说明

## 远程地址

```
CC-Desk  git@github.com:abyss2021/CC-Desk.git (SSH)
```

## 推送命令

```bash
git push CC-Desk main
```

## 大文件注意事项

以下构建产物已加入 `.gitignore`（`CC-Desk/.gitignore`），**不可提交**：

- `windows/CC-Desk-Portable-*.zip`（约 52 MB）
- `windows/portable-windows-x64/`（含 `claude-sidecar.exe` 124 MB）
- `windows/installer-output/`（安装版 exe 约 37 MB）
- `windows/__pycache__/`

提交前务必确认不包含上述文件，否则 GitHub 会拒绝推送。

## 故障排查

### SSH 密钥

密钥位置：`~/.ssh/id_ed25519`
公钥已添加至 GitHub 账户 `abyss2021`。

如遇 `Permission denied (publickey)`，检查密钥是否存在：
```bash
ls ~/.ssh/id_ed25519
```

### HTTPS 不可用

HTTPS 推送大概率被阻断，统一使用 SSH。
